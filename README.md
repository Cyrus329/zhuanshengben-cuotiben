# 专升本错题本

打开 `index.html` 即可使用。上传到 GitHub Pages 后，可以安装到手机桌面。数据默认保存在当前设备本地，也可以点“云同步”保存到 Supabase，让其他手机看到你的最新进度。

主要功能：

- 内置 M001-M004 高数完整错题包，共 169 题，已按新版 Word 文档替换旧题库。
- 错题卡会分区显示题目、错因和防错关键；根号、分式和指数会用更接近数学书写的样式展示。
- 每题已补“完整解析”，默认隐藏；做完题后点“检查解析”才显示步骤和答案。
- 科目固定为高数、计算机、英语。
- 记录题目、知识点、来源、错题日期、错因、答案、解析和标签，默认来源为作业。
- 支持上传题图，图片会压缩后保存在本地数据里。
- 支持搜索、按科目/状态/标签筛选、按复习日期排序。
- 支持记录复习结果，并自动维护待复习队列。
- 支持 JSON 备份导出和导入。
- 支持 Supabase 云同步：你用编辑密码保存，别人通过公开链接只读查看。

## 云同步

1. 先在 Supabase 的 SQL Editor 运行 `supabase-setup.sql`。
2. 打开错题本，点“云同步”。
3. 填公开编号、显示名称、编辑密码，点“保存并开启同步”。
4. 复制公开链接给别人，格式类似：

`https://你的用户名.github.io/wrong-question-organizer/?public=你的公开编号`
