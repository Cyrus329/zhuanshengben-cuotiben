# 专升本错题本

打开 `index.html` 即可使用。数据会保存在当前浏览器本地，可通过页面右上角的导入、导出按钮备份。

主要功能：

- 内置 M001-M004 高数完整错题包，共 169 题，已按新版 Word 文档替换旧题库。
- 错题卡会分区显示题目、错因和防错关键；根号、分式和指数会用更接近数学书写的样式展示。
- 科目固定为高数、计算机、英语。
- 记录题目、知识点、来源、错题日期、错因、答案、解析和标签，默认来源为作业。
- 支持上传题图，图片会压缩后保存在本地数据里。
- 支持搜索、按科目/状态/标签筛选、按复习日期排序。
- 支持记录复习结果，并自动维护待复习队列。
- 支持 JSON 备份导出和导入。
